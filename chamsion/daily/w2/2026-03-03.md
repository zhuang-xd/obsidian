### 代码审查

添加commit钩子
```shell
curl -Lo .git/hooks/commit-msg http://192.168.80.30:8080/tools/hooks/commit-msg
```

触发钩子，打开后直接保存关闭即可
```
git commit --amend
```

### 修改之前的commit

```shell
git rebase -i HEAD~4
```

把pick改成edit
![[Pasted image 20260303105813.png]]

```shell
git commit --amend --author="xiaodong.zhuang <xiaodong.zhuang@chamsion.com>" --no-edit
git rebase --continue
```

把当前分支推送到远端的master分支的代码审核
```shell
git push origin HEAD:refs/for/master
```
