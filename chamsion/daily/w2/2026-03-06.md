### 录制时间固定60秒

眼镜同步时间的操作是异步的，只做了1次同步检测

如果300ms没同步，就是默认的文件名1970-01-01

- `glass_service_impl.cc`
	- OnStartWebService
	- Start
- `uart_service.cc`
	- StartUartService
	- StartHeartbeat
	- HeartbeatThreadMain
	- wifi_handler_->StopRecord();

```cpp
// 增加时间同步重试机制，确保时间同步成功
int sync_retry_count = 0;
const int MAX_SYNC_RETRY = 3;  // 最大重试3次

while (!is_time_synced() && (sync_retry_count < MAX_SYNC_RETRY)) {
	LOGW("Time not synced, retry sync (count: %d)", sync_retry_count + 1);
	
	// 发送时间同步指令
	auto UartFrame = WifiHandler::MakeUartFrame(DEV_ID_MASTER, DEV_ID_APP, APP_CMD_SYNC_TIME, 0);
	if (UartFrame) {
		uart_service_.Uart2Send(UartFrame->header_start, UARTFrame_SIZE(0));
		free(UartFrame);
	} else {
		LOGE("Failed to create sync time frame");
	}
	
	// 延长等待时间，给足够的同步响应时间
	UartHelper::SleepMilliSeconds(500);  // 从300ms延长到500ms
	sync_retry_count++;
	
	// 检查同步状态
	if (is_time_synced()) {
		LOGI("Time sync success after %d retries", sync_retry_count);
		break;
	}
}
```


```cpp
// 生成文件名前再次检查时间同步状态
char file_name[64] = { 0 };
if (!is_time_synced()) {
	// 时间仍未同步时，使用当前时间戳+随机数生成文件名，避免1970-01-01
	LOGW("Time still not synced, generate fallback filename");
	snprintf(file_name, sizeof(file_name), "photo_%lld_%d", 
			 (long long)time(nullptr), rand() % 1000);
} else {
	get_file_name(file_name, sizeof(file_name), PHOTO);
}
```
