## 烧录3303

开启ubuntu的共享文件夹
```shell
sudo vmhgfs-fuse .host:/ /mnt/hgfs -o allow_other
```
### 编译固件

运行 `.cy_copy.sh` 使用老版本的配置，再进行编译
![[Pasted image 20260227141103.png]]
### 烧录

拷贝镜像文件 `g5k6_Flash.img` 到 `tftpd32.exe` 的目录，并打开`tftpd32.exe`
![[Pasted image 20260227142834.png]]

配置uboot局域网
```
set serverip 192.168.1.10
set ipaddr 192.168.1.3
set netmask 255.255.255.0
set gatewayip 192.168.1.1
saveenv
```

通过uboot烧录镜像
```
tftp a1000000 g5k6_Flash.img; sf probe 0; sf erase 0x0 0xc00000; sf write a1000000 0x0 0xc00000
```

![[Pasted image 20260227141156.png]]

输入命令重启主板
```
fastbootcmd 1;save;reset
```
