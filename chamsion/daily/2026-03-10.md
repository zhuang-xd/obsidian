## 配置远程仓库

在root用户目录 `~\.ssh\config` 添加 

```shell
Host 192.168.4.13
    Port 29418
    User Chamsion
    PubkeyAcceptedKeyTypes +ssh-rsa
    HostkeyAlgorithms +ssh-rsa
    IdentityFile /root/.ssh/id_rsa  # 明确指定私钥路径（可选，确保agent能识别
```

```shell
git clone "ssh://Chamsion@192.168.4.13:29418/jl7016g"
git clone "ssh://Chamsion@192.168.4.13:29418/2809rpc_change_content"
```
