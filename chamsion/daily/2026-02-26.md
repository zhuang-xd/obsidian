#### 拍照会暂停播放音乐

在当前的蓝牙协议栈中，手机在进入/操作相机时会通过 HID/AVRCP 发送一组“拍照”按键码。  
库里默认启用了 **自动暂停标志**（`auto_pause_flag = 1`），收到这类命令时会认为是用户想要暂停音乐，于是 A2DP/AVRCP 自动执行暂停操作。

`bluetooth_spp_user.c`
```cpp
case USER_UART_MSG13:
	log_info("收到拍照消息, 取消自动暂停\n");
	__set_auto_pause_flag(0);
	break;
```

#### 使用眼镜听歌过程中录像会暂停音乐播放

`bluetooth_spp_user.c`
```cpp
	static u8 video_pause_flag = 0; // 录像期间暂停音乐的标记

case USER_UART_MSG50:
	printf("----UART MSG50: Record video--\r\n");
	{
		u8 recard_data[8];
		recard_data[0] = 0xAA;
		recard_data[1] = 0x02;
		recard_data[2] = 0x03;
		recard_data[3] = 0x50;
		recard_data[4] = 0x00;
		recard_data[5] = 0x00;
		recard_data[6] = 0x61;
		recard_data[7] = 0x84;
		user_uart2_send(recard_data, 8);
		spp_uart_buf[0] = 8;
		memcpy(&spp_uart_buf[1], recard_data, 8);
		gpio_set_direction(SPP_UART_CTL_PORT, 0);
		gpio_set_output_value(SPP_UART_CTL_PORT, 1);

		// 开始录像后暂停音乐播放
		user_send_cmd_prepare(USER_CTRL_PAUSE_MUSIC, 0, NULL);
		video_pause_flag = 1;
		__set_auto_pause_flag(0);
		break;
	}
	break;

case USER_UART_MSG51:
	printf("----UART MSG51: Stop video--\r\n");
	{
		u8 stop_data[8];
		stop_data[0] = 0xAA;
		stop_data[1] = 0x02;
		stop_data[2] = 0x03;
		stop_data[3] = 0x51;
		stop_data[4] = 0x00;
		stop_data[5] = 0x00;
		u16 crc = crc16_calc(stop_data, 6);
		stop_data[6] = crc & 0xFF;
		stop_data[7] = (crc >> 8) & 0xFF;
		user_uart2_send(stop_data, 8);
		spp_uart_buf[0] = 8;
		memcpy(&spp_uart_buf[1], stop_data, 8);
		gpio_set_direction(SPP_UART_CTL_PORT, 0);
		gpio_set_output_value(SPP_UART_CTL_PORT, 1);

		/* 录像结束后，如果先前因视频而暂停了音乐，则恢复播放 */
		if (video_pause_flag) {
			user_send_cmd_prepare(USER_CTRL_AVCTP_OPID_PLAY, 0, NULL);
			video_pause_flag = 0;
		}
		/* 恢复自动暂停功能 */
		__set_auto_pause_flag(1);
	}
	break;
```


#### 点击拍照会有两次提示音

1. **在耳机端屏蔽本地提示**  
    把 `tone_play_index(IDEX_TONE_PHTOTO,1)` 移除或加条件判断，只在需要时（比如长按、  
    MCU 没给提示时）才播放。
2. **或修改按键处理，不让短按同时生成 SPP 消息**  
    让短按只发 HID，长按才发 [](vscode-file://vscode-app/c:/Users/xiaodong.zhuang/AppData/Local/Programs/Microsoft%20VS%20Code/072586267e/resources/app/out/vs/code/electron-browser/workbench/workbench.html)。这样手机发音后耳机不再收到第二个  
    播放请求。
3. **如果可以在手机端控制**，也可让手机收到 HID 后不要再转发拍照消息给耳机。

#### 固定录制一分钟

`uart_service.cc`
```cpp
void UartService::HeartbeatThreadMain(void) {
    uint64_t now = 0;
    s_last_heartbeat_time = 0;
    uint32_t try_waiting_for_Web = 0;
    while (is_running_) {
      usleep(100 * 1000);

      now = now_time_us();
      if (get_record_state() && record_start_time_ > 0 && (now - record_start_time_ >= base::kRecordTimeMax)) {
        wifi_handler_->StopRecord();
        LOGC("Stop record, now:%llu, record_start_time:%llu, duration:%llu us",
             now, record_start_time_, now - record_start_time_);
      }
}
```
